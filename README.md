# Yarn.solutions

## Online Marketplace For Freelance Services.

<br>

### In the main directory create `config.env` file and paste all environment variables.

### `PORT=`

### `STRIPE_PUBLISHABLE_KEY=`

### `STRIPE_SECRET_KEY=`

### `MONGODB_URI=`

### `SECRET_KEY=`

### `DBUSERNAME=`

### `DBPASSWORD=`

### `DBNAME=`

## Run command

### `npm install`

### `npm run devStart`

### `npm start`
